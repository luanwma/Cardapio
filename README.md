# Projeto de Cardápio


## 📖 Descrição:
Criação de uma API em Spring Boot, utilizando frameworks como JPA, Lombok, e banco de dados PostgreSQL.
No Front-end foi utilizado TypeScript, React e Vite.


## 🚀 Pré-Requisitos

- **Java 17**
- **Maven**
- **Spring Boot**
- **JPA**
- **Lombok**
- **PostgreSQL**
- **TypeScript**
- **Vite**
- **React**
